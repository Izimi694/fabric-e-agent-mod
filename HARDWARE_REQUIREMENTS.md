# 硬件与成本要求

## 服务器配置

| 硬件 | 最低 | 推荐 | 说明 |
|------|:----:|:----:|------|
| CPU | 任意现代 x86 (2 核+) | 4 核+ | 不跑 LLM, 只跑 Minecraft + Mod |
| RAM | 8 GB | 16 GB | Minecraft+Mod 约 4-6 GB, 无 LLM 开销 |
| GPU | 集成显卡 | 任意 | LLM 不走本地推理 |
| 存储 | 1 GB 剩余 | 4 GB+ | 主要为 Minecraft 地图和反射 JSON |
| 网络 | 需要 | 需要 | LLM 走云端 API |

## LLM 推理策略

**E-Agent 不运行本地 LLM 模型。** 所有 LLM 调用通过云端 API 异步完成:

| 提供商 | 推荐模型 | 月成本估算 | 延迟 |
|--------|---------|:---------:|:----:|
| DeepSeek | deepseek-chat | ~$0.3 | 1-3 s |
| OpenAI | GPT-4o-mini | ~$0.5 | 1-2 s |
| Anthropic | Claude Haiku | ~$0.5 | 1-2 s |

**调用频率**: 6-12 次/小时 (每 ~5-10 分钟一次 + 早期预警触发)
**每次预算**: ≤ 1024 tokens
**月 Token 用量**: ~400,000 tokens (输入+输出)
**年成本**: < $5

## 本地 LLM (仅特殊场景)

本地推理仅在以下条件同时满足时可行:

- GPU: RTX 3060 12GB+ 或同等
- 专用 2 个 CPU 核心隔离给 LLM
- 模型: 7B Q4_K_M 量化或更小
- 风险: TPS 可能跌至 < 15, GC 暂停 1-2s

**默认不推荐。** 生产环境走云端 API。

## 启动与缓存

- **启动**: 懒加载 3 个通用预设, < 5 秒
- **缓存**: `.cache/landscapes/*.json` — 按群系/场景缓存, 永不过期
- **磁盘影响**: 缓存 < 10 MB, 反射库 < 1 MB
