# 唯物主义人工生命框架 —— 工程说明（非哲学版）

本文档不是哲学宣言，不是普适真理。
它是 AIPlayerMod 模组 当前架构的工程理由说明。
哲学讨论请左转去读 Dennett / Clark / Chalmers。

学习是为了不学习。思考是为了不思考。
每一份思考都应该产出一份反射。
反射覆盖的场景越多，思考越少。
目标是：思考次数 = O(陌生事件)，不随运行时间增长。

## 一、这个模组解决什么问题

在 Minecraft 里控制一个 AI 玩家，要求：

- **不无限烧钱** → 少调大模型 API
- **能长期在线** → 本地自动学习、自动固化
- **个体差异** → 不同 bot 性格、技能不同
- **行为可解释** → 什么情况做什么，能追溯

这不是在研究"意识本质"，
而是在一个有限、可控、可重复的环境里，
建一个经济且可演化的行为控制系统。

## 二、设计的唯一约束（不是哲学）

### 核心经济约束

**本地能做的事，绝不调 API。**

理由（不是信仰，是账单）：

- API 按量收费
- 网络延迟不可控
- 玩家希望 AI 挂机不花钱

→ 所有设计围绕这一条展开。

### 核心抽象（OS 内核视角）

| 概念 | 工程实现 | 设计参考 |
|------|---------|---------|
| 进程/线程 | 反射（ConditionedReflex） | 条件反射（生物启发） |
| 应用程序 | LLM | — |
| 系统调用 | 12 原子动作（BasicActionAdapter） | — |
| 调度器 | MetaScheduler | — |
| 文件系统 | 反射库（conditioned/*.json） | — |
| 热/冷缓存 | stw/ltb 双权重 | 突触可塑性（参考） |
| 动态优先级 | 激素系统（HormonalSystem） | 神经调节（参考） |
| 进程 fork | 繁衍 / 三规则继承 | 遗传算法 |

所有"设计参考"标注的是灵感来源，不是声称模拟生物学。

## 三、我们实际造了什么（不是空想）

### 1. 六层拦截器（工程分层，不是脑科学）

每一层存在的意义，都是让下一层不需要被调用。

| 层 | 名称 | 做什么 | 成本 | 不学习的理由 |
|----|------|--------|------|-------------|
| L0 | 生存本能（硬编码） | 熔岩 / 虚空 / HP<2 就跑 | 0 | 先天不学 |
| L1 | 先天预警 | 一次示例 → 永久记住（如"creeper 危险"） | 0 | 学一次 / 永远不学 |
| L2 | 条件反射 | 从后果积累权重（stw↔ltb） | 0 | 用进废退 |
| L3 | 模仿学习 | KNN + 贝叶斯 + 前额叶抑制 | 0 | 观察 → 固化 |
| L4 | 自组织 | 随机动作 → 相关性检测 → 自生成反射 | 0 | 乱试学会 |
| L5 | 本地规划 | 规则匹配 + 启发式拆任务 | 0 | 模板拆解 |
| L6 | LLM | 仅全新复杂任务 | 高 | 最后一次思考 |

这不是"脑区映射"，是**成本和不确定性不同的行为来源**。

### 2. 条件反射系统（不是"意识"）

核心数据结构：

```json
{
  "trigger": "看见铁矿石且背包有镐",
  "steps": ["移动", "挖掘", "捡起"],
  "successRate": 0.92,
  "stw": 0.8,
  "ltb": 0.3
}
```

行为规则：

- 成功 → stw ↑
- 失败 → stw ↓
- stw > 阈值 → 自动执行
- successRate < 30% → 休眠，不删除
- 反复成功 → ltb ↑（慢变）

这完全可以解释为**加权有限状态机**。
不需要叫它"条件反射"也可以，叫"经验规则"也行。

### 3. 激素系统（四个计数器，不是情绪）

| 变量 | 触发方式 | 作用 |
|------|---------|------|
| stress | 受伤 / 失败 | 倾向生存行为 |
| aggression | 战斗胜利 | 倾向攻击 |
| curiosity | 发现新奇 | 倾向探索 |
| intimacy | 玩家表扬 / 批评 | 影响社交行为 |

这不是"快乐/悲伤/爱"，
这是**行为倾向的动态调节器**。→ 绑定到 `selectPerspective()` 输出的视角选择，而非直接修改动作。

### 4. 繁衍与正态分布（不是辩证法）

繁衍规则：

- 继承：两 bot 参数取平均 + 噪声
- 独立继承：单 bot 参数减半 + 噪声
- 突变：小概率随机

**数学结果（中心极限定理）：**
群体分布 → 正态分布。

这不是"否定之否定"，
这是随机线性组合 + 噪声的自然结果。

## 四、我们刻意不做的事（不是"做不到"，是"不做"）

### ❌ 不声称"证明了唯物主义"

为什么：

- 一个有限、封闭、人工设计的系统，证明不了任何本体论命题
- 模拟 ≠ 还原

我们只做一件事：
在这个具体系统里，所有行为可追溯到底层数值。
这句话是真的，就够了。

### ❌ 不讨论"自由意志"

因为：

- 这个问题在工程上无操作定义
- 你说"随机初始化就是自由" → 我不争，也不当真理

我们改成可检验的问题：

> 给定相同的初始参数和相同的随机种子，
> 两次运行的行为是否完全一致？
>
> 一致 → 可复现
> 不一致 → 找 bug

这是工程，不是神学。

### ❌ 不引用辩证法三大规律

原因很简单：
把"权重累积 → 跨阈值 → 行为突变"叫作"量变质变"，
不增加任何可操作性。

我们改成：

- 阈值在哪里
- 怎么调
- 为什么选这个值

## 五、我们真正需要回答的工程问题（不是什么哲学问题）

| 问题 | 当前状态 |
|------|---------|
| 本地反射真的能覆盖 90% 场景吗？ | 待验证 |
| 固化阈值为什么选 ≥3 次成功？ | 经验值，可调 |
| 休眠反射恢复机制是否有效？ | Phase 3 测试中 |
| 群体分布真的会正态吗？ | Phase 6 验证 |
| 用户是否感觉到 AI "像一个人"？ | 最终唯一有效指标 |

最后一个问题：

> 玩家玩的时候，觉得这个 AI "活着"吗？

这不是哲学问题，这是**用户研究问题**。

## 六、总结（清晰版）

### 我们做的是：

- 一个 Minecraft AI 模组
- 基于 **成本优先 + 本地固化 + 可演化参数**
- 行为来源 = 6 级分层拦截器（L0-L6）
- 个体差异 = 参数初始化 + 学习历史
- 群体分布 = 随机 + 继承的数学结果

### 我们不做的是：

- ❌ 证明唯物主义
- ❌ 解释意识本质
- ❌ 讨论自由意志
- ❌ 把工程决策包装成哲学真理

## 七、Related Work（对标定位）

### 相关但不相同

| 维度 | 已有工作 | 本系统做法 |
|------|---------|-----------|
| 行为分层 | Brooks Subsumption Architecture (1986) | 6 层拦截器 + 按成本分层 |
| 双权重学习 | TD(λ) (Sutton 1988)；Dual-process RL (Botvinick 2019) | 简化的工程实现 + 休眠机制 |
| 情绪/激素调节 | EM-BDI (2000s)；Homeostatic regulation (Jiang 2021) | 绑定到 Perspective 选择 |
| LLM + 技能复用 | Voyager (2024)；Generative Agents (Park 2024) | 反射固化，成本收敛到 0 |

### 这是组合，不是发明

- 双权重（stw/ltb）≈ Q-learning 的简化变体
- 繁衍 ≈ 遗传算法的特例
- MetaScheduler ≈ Behavior Tree 的变种
- 六层分层 ≈ Subsumption Architecture 的分层继承

### 实验验证（2025年论文）

以下论文从神经科学和计算建模角度独立验证了本架构的核心设计决策：

| 论文 | 核心发现 | 对应本架构 |
|------|---------|-----------|
| Lopez-Ojeda & Hurley (2025) | 习惯(S-R)与目标导向(R-O)分离 | L2 条件反射 vs L6 LLM |
| PNAS (2025) | 海马体检索速度随学习加快5倍 | stw→ltb 固化机制 |
| Nature Human Behaviour (2025) | 奖励学习的本质是工作记忆+习惯，非RL | 不以RL为核心，用进废退筛选 |

这些不是本架构的证据——但它们表明本架构的设计方向与神经科学独立验证的结论一致。

**独特性的可能方向：** "本地优先 + 反射固化 + 休眠不删除"的组合机制——不同于现有 LLM 缓存方案。

### 生物学启发的声明

本系统引用生物学概念（条件反射、激素、突触可塑性）作为设计启发，不是声称模拟神经系统。以下对应关系是工程简化：

- 双权重 (stw/ltb) ⇢ 类似突触可塑性的快/慢成分，但不模拟生物机制
- 激素系统 ⇢ 类似神经调节的行为倾向调节，不是内分泌系统模型
- 休眠/复活 ⇢ 类似记忆再巩固，但纯工程决策

这些不是学术贡献，是工程上的简化。
